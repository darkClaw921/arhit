# ui.ts

**Type:** file  
**Path:** src/commands/ui.ts  
**Updated:** 2026-03-30T18:21:32.771Z

Веб-интерфейс для просмотра архитектуры и документации. uiStartCommand — запускает HTTP-сервер в фоновом процессе (daemon), сохраняет PID в .arhit/ui.pid. uiStopCommand — останавливает сервер по PID. uiStatusCommand — проверяет статус. Встроенный SPA с тёмной темой: навигация по архитектуре, зависимости с фильтрами, Mermaid-диаграммы, поиск по документации.
