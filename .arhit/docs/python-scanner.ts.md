Анализатор Python-кода на основе регулярных выражений (без внешних зависимостей). Извлекает: функции (def), классы (class), переменные верхнего уровня, импорты (from/import), вызовы функций, наследование классов. Приватные элементы (_prefix) помечаются как неэкспортируемые.
